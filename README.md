#  AI Fact-Checker API

هذا المشروع عبارة عن نظام ذكاء اصطناعي للتحقق من الأخبار والادعاءات باستخدام بنية **RAG**.

##  المميزات
- **التحقق اللحظي:** يستخدم Tavily API للبحث في الإنترنت عن أحدث الأدلة.
- **عقل ذكي:** يعتمد على موديل Llama 3.3 عبر Groq لسرعة الاستجابة.
- **دعم الروابط:** القدرة على سحب محتوى المقالات من اللينكات وتحليلها.
- **ثنائي اللغة:** يدعم الرد باللغة العربية والإنجليزية.

##  التقنيات المستخدمة
- **Framework:** FastAPI
- **AI Model:** Llama-3.3-70b (Groq)
- **Search Engine:** Tavily AI
- **Deployment:** Render

##  كيفية التشغيل
1. قم بتحميل المكتبات:
   `pip install -r requirements.txt`
2. قم بتشغيل السيرفر:
   `uvicorn main:app --reload`
